# Milestone citations

Every milestone in the tech tree should have a **proper, verifiable citation** when possible.

## Rules

- **`source`** (optional but strongly recommended): URL to a single primary source for the event.
- Prefer **primary sources**: official announcements, press releases, papers (arXiv, DOI, journal), government/Nobel/agency pages, company blogs.
- Use **specific dates** from that source (publication date, approval date, launch date).
- Do **not** invent milestones or use placeholder dates (e.g. `2025-01-01` for “future trend”); only add events that have a citable reference.
- If no single URL exists (e.g. a diffuse trend), omit the milestone or rephrase as a specific citable event.

## Good `source` examples

- Papers: `https://arxiv.org/abs/...`, `https://www.science.org/doi/...`, `https://www.nature.com/articles/...`
- Official: `https://www.fda.gov/...`, `https://www.nobelprize.org/...`, `https://openai.com/blog/...`
- Company: `https://company.com/news/...`, investor press releases with stable URLs

## Domains (all at same rigor)

- **AI**: OpenAI/Google/Anthropic/DeepMind blogs, arXiv, safe.ai
- **Biotech**: FDA, Nobel, Science/Nature/NEJM, company IR
- **Energy**: NREL, LLNL, IEA, Nobel, company and project sites
- **Quantum**: Nature, IBM/Google/IonQ/D-Wave, NIST, arXiv
- **Nanotech**: Nature, Science, Nobel, company, FDA (Doxil)
- **Space**: NASA, SpaceX, ESA, mission and project sites
- **Neurotech**: Nature, NIH/PubMed, company (Neuralink, Kernel), DARPA, FHI
- **Robotics**: Boston Dynamics, Tesla, Amazon, Nature/Science, company blogs
- **Compute**: Nobel, Nature, IEEE, TSMC/Intel/AMD/NVIDIA, JEDEC
